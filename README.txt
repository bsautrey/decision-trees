---Overview---

Implementation of decision trees from David Sontag NYU machine learning course: http://cs.nyu.edu/~dsontag/courses/ml16/slides/lecture11.pdf.

---Requirements---

* numpy: https://docs.scipy.org/doc/numpy/user/install.html
* matplotlib: https://matplotlib.org/users/installing.html

---Example---

1) Change dir to where decision_trees.py is.

2) Run this in a python terminal:

from decision_trees import DecisionTrees
dt = DecisionTrees()
dt.generate_example_1()

OR

from decision_trees import DecisionTrees
dt = DecisionTrees()
dt.generate_example_2()

OR

See the functions generate_example_1() and generate_example_2() in decision_trees.py
